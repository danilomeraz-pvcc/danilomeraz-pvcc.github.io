    Operating Systems Terminology

    Terms Definitions
    Operating Systems The software that controls the pverall operation of the computer including file management, the user interface, and the program execution environment.
    Job Queue A data structure maintained by a job scheduler software containing jobs to run; An ordered list of jobs waiting to be processed by a subsystem.
    FIFO Stands for First in, First out, which is an accounting method that is used to manage and value assets.
    Interactive Processing Refers to the process of interacting directly with a program and/or data for immediately modifying, retrieving and/or displaying information.
    Time-Sharing A type of operating system that allows multiple users or processes to share the same system resources in a time-sliced manner.
    Multiprogramming A technique that allows multiple programs to run concurrently on a single computer system.
    Application Software A computer program that is designed to handle and assist specific tasks for users.
    System Software A type of computer program that manages a computer's hardware and provides services to higher-level software.
    Utility Software A type of software designed to help analyze, configure, optimize ro maintain a computer; It provides generic services to the computer.
    Graphical User Interface (GUI) A form of user interface that is a way of interacting with a computer using items such as windows, icons, and menus.
    Shell A program that provides an interface between the user and the operating system.
    Kernel Refers to the most basic level of an operating system of a computer. It's responsible for resource allocation, file management, and security.
    Device Driver A software program that enables a computer's operating system to communicate with a hardware device.
    File Manager A software program that assist users manage files and folders on their computer.
    Memory Manager A software utility that works in conjunction with the operating system to manage memory more efficiently.
    Virtual Memory A technique used by operating systems to increase the capacity of its main memory.
    Boot Strapping Refers to the technique of writing the compiler in the source programming language that it intends to compile.
    ROM (Read-Only Memory) A type of storage that permanently stores data on PCs and other electronic devices.
    Deadlock A situation where two or more processes are not able to proceed because each is waiting for a resource that is being held by another process.
    Spooling The process of copying data between different devices, such as a computer and a printer.
    Super User (or administrator) A user of a computer system with special priveleges needed to administer and maintain the system.
    Multi-factor authentication An electronic authentication method that requires users to provide two or more pieces of evidence to an authentication mechanism to gain access to a website or application.
